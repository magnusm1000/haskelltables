# Data Repository

This repository contains the raw CSV datasets used for the analysis and evaluation within the thesis **Developing and Evaluating an AI-Based Exercise Platform for an Introductory Functional Programming Course**.

---

## Files

* **tracking.csv**
    Contains the quantitative user activity data: total time spent in exercises, number of exercises solved, compilation attempts, success rates etc.  broken down by user.

* **formresponses.csv**
    Stores the results from the final user survey including written feedback.
  
* **recursion.csv**
    Contains the exercise data (description, snippet, tests) and manual quality ratings for all generated exercises on the topic of **Recursion**.

* **hof.csv**
    Contains the exercise data and manual quality ratings for all generated exercises on **Higher-Order Functions (HoF)**.

* **types.csv**
    Contains the exercise data and manual quality ratings for all generated exercises on **Types and Algebraic Data Types**.

* **monad.csv**
    Contains the exercise data and manual quality ratings for all generated exercises on **Monads**.

* **eval (1).csv**
    Contains the exercise data and manual quality ratings for all generated exercises on **Evaluation Strategies**.
